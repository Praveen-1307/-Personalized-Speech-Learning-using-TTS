# Implementation Specification: Emotion Detector (`emotion_detector.py`)
*(Simple Explanation Version)*

## 1. Overview
The Emotion Detector acts as the "Psychological Vibe Checker." Its purpose is to predict if a user sounds happy, sad, angry, or calm based on a 5-second recording.

## 2. The Problem It Solves
How can a computer tell if someone is depressed or energetic? A computer does not understand language or tone; it only understands numbers.

## 3. How It Works in the Code (The Tricks)
1. **Classical Machine Learning:** It uses an old-school machine learning model called a Support Vector Machine (SVM), rather than a massive Deep Learning neural network.
2. **Modeling the Throat:** The code looks at 13 specific numbers generated by the Feature Extractor called "MFCCs". These numbers mathematically map the physical shape of the speaker's throat, tongue, and mouth while they are speaking.
3. **Classification:** We trained the SVM to recognize these shapes. The code feeds the 13 numbers into the SVM. If the SVM recognizes that the numbers look like "Shape A", it knows the person is yelling and outputs the string "Angry". If they look like "Shape B", it knows the person is crying and outputs "Sad".
